# Berg

Berg is a Kibana-style web UI for AWS S3 Tables, built on top of an Athena query
client and the Glue Data Catalog. It is a fork of
[HyperDX](https://hyperdx.io)/ClickStack repurposed for analytical query
workflows on top of S3 Tables.

This is a private/in-development project. Documentation and deployment guides
are forthcoming.
